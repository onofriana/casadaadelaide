# Casa da Adelaide

**Onofriana apresenta a Casa da Adelaide** — portal de fado para crianças.

> O fado tem raízes. Aqui crescem.

---

## Estrutura

| Ficheiro | Página |
|---|---|
| `index.html` | Entrada — apresentação e navegação |
| `colorir.html` | Imagens ASCII para imprimir e colorir |
| `quiz.html` | Quiz sobre fado — 10 perguntas |
| `sopa.html` | Sopa de letras interactiva |
| `poema.html` | Construtor de quadra de fado |
| `sala.html` | Sala de vídeo curada pela Onofriana |
| `workshop.html` | Fichas de workshop para download |
| `escola.html` | Página institucional — escolas, câmaras, famílias |

## Publicar no GitHub Pages

1. Ir a **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` / `root`
4. Domínio personalizado: configurar em DNS quando disponível
